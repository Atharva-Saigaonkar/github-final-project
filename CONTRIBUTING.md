# Reporting Bugs

### Before Submitting a Bug Report

Please ensure:

- You are using the latest version  
- You have checked documentation and existing issues  
- You have searched online  
- You have collected necessary information:
  - OS / platform  
  - Steps to reproduce  
  - Logs or stack traces  
  - Versions of related tools  

### Submitting a Good Bug Report

- Open a new [Issue](/issues/new)  
- Provide clear expected vs actual behavior  
- Include reproducible steps  
- Add screenshots or logs if possible  

---

# Suggesting Enhancements

Before submitting an enhancement:

- Confirm it's not already implemented  
- Search existing issues  
- Ensure your suggestion aligns with project goals  

When submitting a suggestion:

- Use a clear title  
- Describe the enhancement in detail  
- Provide reasoning and justification  
- Include examples, screenshots, or references
